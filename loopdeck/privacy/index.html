LoopDeck Privacy Policy

Last updated: May 21, 2026

Introduction

LoopDeck is a Chrome extension created by Unclocked Studios LLC to help users create screen profiles and automatically cycle through dashboard URLs. This Privacy Policy explains how LoopDeck handles data.

Data Collection

LoopDeck does not collect, sell, share, or transmit personal data, browsing history, dashboard URLs, website content, analytics, or usage activity to Unclocked Studios LLC or to third parties.

Local Storage

LoopDeck stores extension settings locally in the user's browser using Chrome extension storage. This may include screen profile names, dashboard nicknames, dashboard URLs, timer settings, and feature preferences. This data is used only to provide LoopDeck's dashboard cycling features.

Site Access

LoopDeck may request site access for dashboard URLs selected by the user. This access is used only to run dashboard timers, Smart Pause, tab cycling, fullscreen behavior, and related user-facing features. LoopDeck does not use this access to collect or transmit website content.

Payments

LoopDeck uses ExtensionPay and Stripe to process payments for the Unlimited plan. Payment processing is handled by those services. LoopDeck does not receive or store credit card numbers or payment details.

Data Sharing

LoopDeck does not sell, share, or transfer user data.

Changes to This Policy

We may update this Privacy Policy from time to time. Any changes will be posted on this page with an updated effective date.

Contact

If you have questions about this Privacy Policy, contact us at support@unclocked.app.
