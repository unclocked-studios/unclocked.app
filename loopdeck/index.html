Common Questions

Does LoopDeck collect browsing history?

No. LoopDeck stores screen profiles, dashboard URLs, timers, and preferences locally in Chrome storage.

Why does LoopDeck request site access?

Site access is requested only for dashboard sites in the selected screen profile. It allows LoopDeck to run timers and Smart Pause on those dashboard tabs.

How does Smart Pause work?

Smart Pause temporarily suspends cycling when someone interacts with a dashboard, then resumes after the idle timer expires.
